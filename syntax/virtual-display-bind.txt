Bind virtualbox to virtual display
  Right WinKey Fn left arroy (HOME) > Devices > View > Video Screen 1 \
  > Use Host Screen 1
    Use Host Screen 2
    Use Host Screen 3

To map virtualbox to a virtual display bring of hidden menu with
HostKey + Home on GCI Network Services, covellite the HostKey is mapping
to Right WinKey so the hidden menu is available
via Right WinKey Fn left-arrow (Home) > View > Video Screen 1 \
  > Use Host Screen 1
    Use Host Screen 2
    Use Host Screen 3

on the HHK keyboard.
